# Conversational Data Analyst (RAG)

## Overview
This project is an AI-powered conversational data analysis system that allows users to query datasets using natural language.

## Technologies Used
- Python
- Streamlit
- FAISS
- Sentence Transformers
- LLM (Claude / RAG)

## Features
- Ask questions in plain English
- Automatic data analysis
- AI-generated insights
- Interactive visualizations

##  How to Run
```bash
python -m streamlit run app.py
